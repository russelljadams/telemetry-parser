# Spec Sheet: Reporting

## Purpose

Generate human-readable summaries for each session and weekly rollups.

## Outputs

- Markdown session report
- Weekly rollup report

## Contents

- Session metadata
- Lap time stats
- Incident summary
- Error event summary
- Hotspots by lap distance
- Recommended focus areas

## Non-Goals

- Real-time dashboards
- Sponsor-ready graphics (future)
